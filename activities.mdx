**Ever wondered who uses a query or when it was last accessed?**

**Activities** provide social proof for your queries, giving you visibility into your personal or team queries. 
<br></br>You can see who interacted with the query and when it was last accessed.

<img src="/images/sherloq_activities.png" 
   alt = "Activities"
   width="300" />

Each interaction is tracked and includes one of the following actions performed on the query:

- Viewed the query or entered the query profile
- Used the query as a snippet
- Copied the query
- Shared the query
- Updated the query

<Check> We prioritize commonly used queries in the search results. </Check>

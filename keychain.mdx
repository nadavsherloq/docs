How to allow permissions when the message \*\*“Microsoft Workplace Join Key”\*\* appears:

<Steps>
  <Step title="Open Keychain Access">
    Press command+space and type "**keychain access**" select
  </Step>

  <Step title="Once in Keychain Access, while in tab 'All items', type in search 'join'" />

  <Step title="Delete everything that comes up" />

  <Step title="After go to Sherloq plugin and log in again into Microsoft account" />
</Steps>

<Tip>
  &#x20; This  won’t impact anything else, including other security and privacy configurations.
</Tip>

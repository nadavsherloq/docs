---
title: 'Filtering'
---

Our search offers several ways to filter query results:

- **By Creator:** Filter by the person who created the query
- Folder: Filter by the folder containing the query
- **Tag:** Filter by the tag assigned to the query
- **Date Added:** Filter by the date the query was created
- **Source:** Filter by the platform where the query was saved (from the IDE or through the Sherloq platform)
